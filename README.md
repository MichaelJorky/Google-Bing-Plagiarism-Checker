# Google & Bing Plagiarism Checker

**Google & Bing Plagiarism Checker** adalah aplikasi desktop khusus **Windows** yang digunakan untuk mengecek tingkat keaslian teks dengan membandingkannya langsung melalui **Google Search** dan **Bing Search**. Aplikasi ini dirancang untuk membantu pengguna memastikan konten tetap original sebelum dipublikasikan.

## âš™ï¸ Cara Menggunakan Aplikasi

1. Pastikan **Microsoft Edge sudah terinstal** di Windows Anda.
2. Jalankan aplikasi **Google & Bing Plagiarism Checker**.
3. Masukkan teks ke kolom **Content to Check**.
4. Pilih **Data Provider**:

   * Google Search
   * Bing Search
5. Klik tombol **Check Plagiarism**.
6. Lihat hasil pengecekan pada kolom **Plagiarism Report**:

   * Teks mirip akan di-highlight
   * Sumber URL ditampilkan
   * Skor keunikan ditampilkan dalam persentase
